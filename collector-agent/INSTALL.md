# 디직스 카운터 수집기 설치 안내 (고객용)

## 📥 받은 파일
**digix-collector.zip** — 약 22 MB

## 🔧 설치 (3단계, 약 1분)

### 1) 압축 풀기
ZIP 파일을 원하는 폴더에 풀어주세요. 권장 위치:
```
C:\HanbyeolCollector\
```
또는 바탕화면 / 다른 위치 자유.

압축 풀면 다음 구조:
```
digix-collector\
├── digix-collector.exe   ← 이 파일을 실행
└── _internal\               ← 프로그램 의존성 (건드리지 않음)
```

### 2) 첫 실행
`digix-collector.exe` 더블클릭.

처음 한 번만 입력 창이 뜹니다:
- **업체명**: 거래처 이름 (예: "디직스코리아")
- **페어링 코드**: `digix` (이미 채워져 있음)

[연결] 버튼 → "✅ 디직스 서버에 정상 연결되었습니다" 메시지.

### 3) 자동 실행 등록 (선택)
화면 우측 하단 트레이에 디직스 아이콘이 떠 있습니다.
**우클릭 → "Windows 시작 시 자동 실행"** 체크하면 PC 켤 때마다 자동 시작.

## 🛡 백신(V3 등) 차단/삭제 시

코드 서명이 안 된 EXE 라 일부 백신이 거짓양성으로 차단할 수 있습니다.

### V3 예외 등록
1. V3 우상단 메뉴 → 환경설정
2. **PC 보안** → 검사 제외 폴더/파일
3. **폴더 추가** → 압축 푼 폴더 (예: `C:\HanbyeolCollector\`)
4. 확인 → 재실행

### 윈도우 디펜더 예외 등록
1. 설정 → 개인 정보 및 보안 → Windows 보안
2. 바이러스 및 위협 방지 → 설정 관리
3. 제외 → **제외 항목 추가** → 폴더 → 압축 푼 폴더 선택

## 🔍 작동 확인

트레이 아이콘 **우클릭** → **"지금 자동 수집 업로드"** 클릭.

5~10초 후 풍선 알림:
```
✅ 디직스 수집기 업로드 완료
장비 N대 · 카운터 M건 전송됨
```

이 메시지가 보이면 정상 작동 중. 이후 **5분마다 자동** 으로 카운터를 디직스 서버로 전송합니다.

## 🆘 트러블슈팅

| 증상 | 확인 |
|---|---|
| 더블클릭해도 아무 반응 없음 | 백신이 EXE 삭제? 위 예외 등록 |
| "지금 업로드" → "0대 발견" | 같은 LAN 에 프린터 있는지, 프린터 SNMP 활성 여부 확인 |
| 풍선 알림 안 보임 | 트레이 우클릭 → "업로드 시 알림 표시" 체크 |
| 페어링 화면 다시 띄우고 싶음 | `%APPDATA%\digix-collector\config.json` 의 `"token"` 줄 삭제 후 재실행 |

## 📁 데이터/로그 위치
```
%APPDATA%\digix-collector\
├── config.json   ← 설정 (token, 업체명, 페어링 코드)
└── agent.log     ← 실행 로그 (오류 추적용)
```
