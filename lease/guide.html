임대관리 사용방법

임대 거래처·장비·카운터·청구를 다루는 실무 안내서입니다. 처음 쓰시는 분은 2번 매달 하는 일부터 보세요.

바로가기 1. 메뉴 구성 2. 매달 하는 일 3. 임대거래처 4. 임대장비관리 5. 임대카운터 6. 추가요금청구 6-2. 임대명세표/계산서 7. 정기방문관리 8. 자료실·사용자·현황 9. 헷갈리는 것 모음

1. 메뉴 구성 어떤 화면이 무슨 일을 하나

메뉴하는 일
임대현황전체 자산·매출을 한눈에 보는 분석 화면 (조회 전용, 입력 없음)
임대장비관리고객 PC의 수집기가 올려주는 실시간 카운터·토너 확인, 수집기를 거래처에 연결
임대거래처모든 설정의 출발점. 거래처·자산 등록, 임대료·무료매수·단가 설정, 수리/판매 내역
임대카운터월별 카운터 입력(수기·엑셀), 추가사용료 자동계산, 이상치 확인
임대추가요금청구고정 임대료 + 초과 사용료를 합쳐 청구서 생성·발송 처리
임대명세표/계산서세금계산서·거래명세표 발행 (금액은 추가요금청구와 동일)
정기방문관리방문 주기 등록 → 다음 방문일 자동계산, D-day 관리
자료실공지·자료·메모용 게시판
사용자관리계정 추가·수정·삭제 (관리자만)
기억할 원칙 하나 임대료·무료매수·단가 같은 기준값은 전부 「임대거래처」에서만 정합니다. 카운터·청구 화면은 그 값을 가져다 계산만 합니다.

2. 매달 하는 일 월 업무 흐름

① 카운터 모으기▶ ② 임대카운터에 입력▶ ③ 이상치 확인▶ ④ 청구서 생성▶ ⑤ 발송 처리
  1. 카운터 모으기 — 수집기를 쓰는 거래처는 임대장비관리 → 📥 카운터 엑셀로 받으면 됩니다. 수집기가 없으면 직접 검침한 값을 씁니다.
  2. 임대카운터에 입력 — 받은 엑셀을 업로드하거나, 당월COUNT 칸을 직접 눌러 입력합니다.
  3. 이상치 확인 — 상단 이상치 경고 카드를 눌러 목록을 보고, 검침이 잘못된 게 없는지 봅니다.
  4. 청구서 생성 — 임대추가요금청구에서 대상 월을 고르고 청구서를 만듭니다.
  5. 발송 처리 — 팩스/카톡/이메일로 보낸 뒤 발송 처리를 눌러 기록을 남깁니다.
가장 많이 헷갈리는 것 — 쓴 달과 청구하는 달은 1개월 차이입니다 4월에 쓴 사용량은 5월에 청구합니다. 청구 화면에서 고르는 월과 실제 데이터 월이 다르게 보이는 건 정상입니다.

3. 임대거래처 모든 기준값을 정하는 곳

거래처 등록

+ 거래처 추가를 누르고 사업자상호를 반드시 채웁니다. 나머지는 아는 만큼만 넣어도 됩니다.

  • 거래처상호가 비면 사업자상호를, 설치주소가 비면 사업자주소를 자동으로 씁니다.
  • 청구일(1~31일)과 결제방법을 넣어두면 청구할 때 그대로 쓰입니다.

자산(임대 물건) 추가

거래처를 고른 뒤 자산을 추가합니다. 품목과 월 임대료가 핵심입니다. 같은 모델 여러 대는 수량으로 한 번에 넣을 수 있습니다.

복사기·프린터 같은 출력기기는 아래 값을 꼭 채워야 카운터 요금이 계산됩니다.

항목뜻
무료 BW / 무료 컬러매달 공짜로 주는 흑백·컬러 장수. 이걸 넘긴 만큼만 돈을 받습니다.
단가 BW / 단가 컬러무료 장수를 넘겼을 때 1장당 받는 금액
흑백/컬러 합산 청구체크하면 흑백·컬러를 합쳐서 계산합니다. 체크해야 아래 두 칸이 열립니다.
기본 무료 매수 / 매수당 단가합산 청구일 때 쓰는 무료 장수와 단가
단가를 바꿀 땐 값을 덮어쓰지 마세요 기존 단가 칸을 고치면 지난 달 청구까지 같이 틀어집니다. 반드시 「단가/기본매수 변경 이력」에 적용일과 함께 새 값을 넣으세요. 그래야 적용일 이전은 옛 단가로, 이후는 새 단가로 계산됩니다.

무상수리내역 / 유상판매수리내역

카드넣는 항목금액 부호
🛠 무상수리내역출장 / 여분토너 / 부품교체자동 마이너스(−)
💰 유상판매수리내역판매 / 수리자동 플러스(+)
금액은 그냥 숫자만 넣으세요 마이너스를 직접 입력할 필요 없습니다. 무상 카드에 넣으면 자동으로 지출(−), 유상 카드에 넣으면 자동으로 수익(+)으로 잡힙니다.

품목 옆에 파란 ASMS 배지가 붙은 줄은 접수관리툴에서 자동으로 넘어온 건입니다(직접 적은 게 아님). 마우스를 올리면 접수번호가 보입니다.

월 합계 카드

최근 12개월치 월 임대료 / 유상판매 / 무상수리 / 합계를 한 줄씩 보여줍니다. 이 거래처가 실제로 남는 장사인지 여기서 바로 확인할 수 있습니다.

만기 거래처는 기본 목록에 안 보입니다 목록 위 활성 거래처 / 만기 거래처 선택을 만기로 바꿔야 보입니다. "거래처가 사라졌다"면 대부분 이 경우입니다.

4. 임대장비관리 실시간 카운터와 수집기 연결

고객 PC에 설치한 수집기가 프린터 카운터와 토너 잔량을 자동으로 올려줍니다. 이 화면은 그 결과를 보는 곳입니다.

상단 3개 숫자 — 눌러서 걸러 보기

온라인 오프라인 매핑대기 — 각각 눌러서 그 상태만 볼 수 있습니다. 다시 누르면 해제됩니다.

상태뜻
온라인최근에 정상적으로 값을 보내온 장비
오프라인30분 넘게 소식이 없는 장비 (PC를 껐거나 프린터가 꺼진 경우가 대부분)
매핑대기어느 거래처 것인지 아직 지정 안 된 장비 → 매핑이 필요합니다

신규 매핑 (수집기를 거래처에 연결)

새 거래처에 수집기를 설치하면 위쪽에 "신규 수집기 연결됨" 배너가 뜹니다. 🔗 신규 매핑 또는 매핑하기 →를 눌러 각 PC가 어느 거래처인지 골라 저장하면 끝입니다.

  • 매핑을 해야 그 PC가 보내는 카운터가 그 거래처 것으로 잡힙니다.
  • 거래처 목록은 입력할수록 좁혀집니다(한 번에 최대 20개만 보임). 회사명을 더 입력해 보세요.

그 밖의 버튼

  • 📥 카운터 엑셀 — 실시간 카운터를 엑셀로 받습니다. 임대카운터에 그대로 업로드할 수 있는 형식입니다.
  • ⬇ 프린터자동설치프로그램 — 고객 PC용 프린터 드라이버 자동설치 프로그램
  • 자산번호·거래처는 표에서 칸을 눌러 바로 수정할 수 있습니다.
우리 제품이 아닌 장비가 섞여 보일 때 해당 장비를 숨김 처리하면 다음에도 다시 나타나지 않습니다.

5. 임대카운터 월별 카운터 입력과 계산

방법 ① 직접 입력

위쪽에서 대상 월을 고른 뒤, 표의 당월COUNT 칸을 눌러 숫자를 넣으면 자동 저장됩니다. 나머지 칸은 전부 자동으로 계산됩니다.

방법 ② 엑셀 업로드

📥 엑셀 자동 입력을 열고 파일을 끌어다 놓습니다(.xlsx / .xls / .csv).

  1. 업로드하면 미리보기가 뜹니다. ✔ 매칭 ⚠ 확인필요 ✖ 미매칭 개수를 확인하세요.
  2. 문제가 없으면 💾 최종 저장을 누릅니다.
  3. 이미 그 달 값이 있으면 중복 제외 후 저장 / 전체 덮어쓰기 중에 고르라고 물어봅니다.
⚠ 확인필요·✖ 미매칭 줄은 저장되지 않습니다 ✔ 매칭된 줄만 저장됩니다. 저장 후 그 거래처가 여전히 "입력 안된 업체"에 남아 있다면 매칭이 안 된 것이니 직접 입력해 주세요.

매칭은 자산번호 → 시리얼 → 회사명+모델 순으로 찾습니다. 자산번호가 가장 정확하니 엑셀에 자산번호가 들어 있으면 제일 좋습니다.

파일 이름의 월과 데이터 월이 1개월 다릅니다 ..._2026_05_01-... 처럼 5월 1일자로 받은 파일은 4월 데이터로 들어갑니다. 저장 전에 대상 월이 맞는지 꼭 확인하세요.

계산은 이렇게 됩니다

월카운터 = 당월COUNT − 전월COUNT
추가카운터 = 월카운터 − 기본매수 (마이너스면 0)
추가사용료 = 추가카운터 × 추가사용단가

기본매수·추가사용단가는 임대거래처에서 정한 값이라 이 화면에서는 못 고칩니다.

위쪽 5개 숫자 — 눌러서 목록 보기

다섯 개 모두 누르면 해당 업체 목록이 아래에 펼쳐집니다.

  • 이상치 경고 — 아래 셋 중 하나에 걸린 경우입니다.
    • 최근 6개월 평균보다 3배 넘게 늘었다 (과다 사용 의심)
    • 전월과 당월이 똑같다 (검침 누락 의심)
    • 당월이 전월보다 줄었다 (장비 교체·초기화 의심)
  • 카운터 오버 — 추가요금이 발생한 업체. 여기서 할인 금액을 직접 입력할 수 있고, 그 할인이 청구서에 그대로 반영됩니다.

6. 임대추가요금청구 청구서 만들고 보내기

먼저 알아둘 것 — 데이터월과 청구월 4월 사용분 → 5월 청구입니다. 화면에 데이터월이 따로 표시되니 항상 확인하세요.

청구금액은 이렇게 나옵니다

① 고정 임대료 = 월 임대료 × 개월수
② 초과 사용료 = 무료 제공 장수를 넘긴 만큼 × 단가 (출력기기만)
③ 청구 합계 = ① + ② − 할인  (할인은 임대카운터에서 입력한 값)
부가세는 별도입니다 청구서의 합 계 (VAT별도)는 부가세가 포함되지 않은 금액입니다.

청구 주기

거래처마다 월별 / 3개월 / 6개월 / 1년으로 다를 수 있습니다. 이번 달이 청구월이 아닌 거래처는 흐리게 표시되고 일괄 발송에서 빠집니다. 정상입니다.

발송 처리

  1. 대상 거래처를 체크하고 선택 항목 일괄 발송을 누릅니다.
  2. 📠 팩스 ✉️ 이메일 등 보낸 방법을 고릅니다.
  3. 보낸 뒤 발송 처리를 누르면 발송됨 상태로 기록됩니다.
시스템이 대신 보내주지는 않습니다 실제 팩스·메일 전송은 직접 하시고, 이 화면은 "보냈다는 기록"만 남깁니다. 이메일을 고르면 메일 작성창이, 카카오톡을 고르면 내용이 복사됩니다.
  • 이미 발송됨·입금완료인 건은 다시 일괄 발송되지 않습니다(중복 방지).
  • 잘못 보냈으면 발송 취소로 초안 상태로 되돌릴 수 있습니다.

6-2. 임대명세표/계산서 세금계산서·거래명세표 발행

거래처에 보낼 세금계산서와 거래명세표를 만드는 곳입니다.

금액이 어디서 오는지 (중요)
  • 임대료(기본금액) → 임대거래처의 「임대 물품 내역」 총금액(청구월분). 그래서 임대거래처에서 보는 금액과 항상 같습니다.
  • 흑백·컬러 추가요금과 할인 → 임대추가요금청구의 전월 사용분. 추가요금이 없는 거래처는 임대료만 나옵니다.
공급가액 = 임대료 + 추가요금 − 할인 (세액은 10%, VAT 별도)
추가요금은 익월청구입니다 위에서 고르는 청구월 기준으로, 6월에 발생한 추가요금은 7월에 청구됩니다.
예) 7월 청구서 = 7월 임대료 + 6월 추가요금·할인. 화면 상단에 "임대료 O월 + 추가요금 O월 사용분"이 표시되니 확인하고 발행하세요.

누가 대상인가

임대거래처의 발행구분에 체크한 거래처가 자동으로 나옵니다.

  • 세금계산서 체크 → 세금계산서 생성 (배지 붉은색)
  • 거래명세표 체크 → 거래명세표 생성 (배지 파란색)
  • 둘 다 체크하면 둘 다 만들어집니다.

쓰는 순서

  1. 대상 월을 고르고 🧮 명세표/계산서 생성·갱신을 누릅니다.
  2. 임대료·카운터·할인이 바뀌어 다시 누르면 금액이 갱신되고, 바뀐 줄은 노란색(💛)으로 표시됩니다.
  3. 청구일·문서종류로 걸러 보고, 보낼 것을 체크합니다.
  4. 📥 세금계산서 발행 또는 📥 거래명세표 발행으로 엑셀을 받습니다(따로 발행).
