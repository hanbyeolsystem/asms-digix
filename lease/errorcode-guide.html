가장 빠른 방법 — 코드만 입력

프린터 화면에 뜬 코드를 그대로 검색창에 입력하면 됩니다. 예를 들어 C2557, E000, SC552 처럼요.

2. 결과 보는 법

목록에서 코드를 누르면 상세 정보가 열립니다.

항목내용
원인이 에러가 왜 발생했는지
해결방법어떤 순서로 조치하면 되는지
팁현장에서 유용한 참고사항 (있는 코드만)
원문제조사 원문 표기 — 매뉴얼과 대조할 때 사용
해결방법은 순서대로 시도하세요 보통 간단한 조치부터 적힙니다. 앞의 방법으로 해결되면 뒤 작업(부품 교체 등)은 하지 않아도 됩니다.

3. 현장 활용 요령

  • 휴대폰에서 그대로 됩니다. 현장에서 코드를 보고 바로 검색하세요.
  • 관리툴 좌측 🔧 에러코드/토너검색 메뉴로 들어가거나, ↗ 새 창을 눌러 크게 볼 수 있습니다.
  • 접수 처리와 함께 쓰기 — 접수 상세에서 처리 내용을 적을 때, 검색한 원인·조치 내용을 그대로 옮겨 적으면 이력이 정확해집니다.
  • 사진을 함께 남기세요 — 접수 화면에서 고장 부위 사진을 첨부하면 다음에 같은 증상이 왔을 때 도움이 됩니다.
토너 검색도 여기서 기종명으로 검색하면 해당 모델 관련 정보를 함께 찾을 수 있습니다.

4. 자주 묻는 질문

이런 경우이렇게 하세요
검색해도 결과가 없습니다 ① 제조사·카테고리 조건이 걸려 있는지 확인하고 조건 초기화 ② 코드 일부만 넣어 다시 검색 ③ 그래도 없으면 한별시스템에 알려주세요. 추가해 드립니다.
같은 코드가 여러 개 나옵니다 제조사마다 같은 코드를 다른 의미로 쓰기 때문입니다. 제조사를 선택하면 정확한 것만 나옵니다.
해결방법대로 했는데 안 됩니다 같은 코드라도 기종에 따라 원인이 다를 수 있습니다. 모델명으로 다시 검색해 보시고, 안 되면 접수 등록 후 방문 조치하세요.
