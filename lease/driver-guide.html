⬇ 프로그램 받기 아래 버튼을 누르면 내려받아집니다. 고객 PC에서 실행하세요. H_Driver 다운로드

1. 프린터 설치 드라이버를 찾아 자동으로 설치

① 프로그램 실행▶ ② 프린터 찾기▶ ③ 종류 선택▶ ④ 설치

네트워크 프린터 (사무실 대부분)

  1. H_Driver.exe를 실행합니다. 설치 과정 없이 바로 열립니다.
  2. 같은 네트워크의 프린터를 자동으로 찾아서 목록에 보여줍니다.
  3. 설치할 프린터를 고르면 모델에 맞는 드라이버가 자동으로 선택됩니다.
  4. 컬러 흑백 팩스 중 필요한 것을 체크합니다. 여러 개를 한 번에 설치할 수 있습니다.
  5. 설치를 누르면 끝날 때까지 진행 상황이 표시됩니다.
프린터가 목록에 안 보이면 프린터 전원과 랜선을 확인하고 다시 찾기를 눌러보세요. 그래도 없으면 프린터 화면에서 IP 주소를 확인해 직접 입력하면 됩니다.

USB로 연결한 프린터

USB 케이블로 연결한 프린터는 USB 연결 쪽에서 고르면 됩니다. 나머지는 네트워크와 같습니다.

같은 이름의 프린터가 이미 있으면 지우고 새로 설치합니다 기존 설정이 꼬여 있을 때 깨끗하게 다시 깔리므로, 재설치 용도로도 그냥 쓰시면 됩니다.
드라이버가 없는 모델이라면 제조사 다운로드 페이지로 바로 가는 버튼이 있습니다. 내려받은 드라이버는 한별시스템에 알려주시면 목록에 추가해 드립니다.

2. 스캔 스캔해서 원하는 곳에 바로 저장

복합기에서 스캔한 문서를 PC·공유폴더·FTP 중 원하는 곳으로 바로 받습니다.

저장 위치 3가지

저장 대상설명
내 PC바탕화면 → scan 폴더에 저장됩니다. 스캔이 끝나면 폴더가 자동으로 열립니다.
공유폴더사내 공유폴더로 바로 저장. 📁 찾아보기로 폴더를 고르면 됩니다.
FTPNAS 등 FTP 서버로 저장. 🔍 찾기를 누르면 서버를 자동으로 찾아줍니다.

설정 방법 (처음 한 번만)

  1. 스캔 화면에서 저장 대상을 고릅니다.
  2. 공유폴더·FTP를 골랐다면 경로(또는 주소·아이디·비밀번호)를 넣고 연결테스트로 확인합니다.
  3. 컬러/흑백, PDF/JPG, 해상도, 양면 등을 필요에 맞게 정합니다.
  4. 한 번 저장해두면 다음부터는 스캔 버튼만 누르면 됩니다.
바탕화면에 「스캔」 바로가기가 생깁니다 처음 실행할 때 자동으로 만들어집니다. 이 바로가기로 열면 스캔 화면만 나와서, 직원분들은 스캔 버튼만 누르면 됩니다.
FTP 저장 시 컴퓨터별로 폴더를 나눌 수 있습니다 공통 폴더는 모두 같은 곳에, 컴퓨터별은 PC마다 폴더가 자동으로 만들어집니다. 여러 대가 함께 쓸 때 섞이지 않습니다.
전송이 실패해도 스캔 파일은 남습니다 네트워크 문제로 전송이 안 되면 PC에 저장해두고 안내해 줍니다. 파일이 사라지지 않습니다.

3. 카운터 수집 사용량 자동 전송

📊 카운터 수집기 실행을 누르면 프린터 사용량(흑백·컬러 매수)을 읽어 자동으로 전송합니다. 따로 설치할 필요 없이 프로그램 안에 들어 있습니다.

  • 수집된 카운터는 관리툴의 임대장비관리에서 실시간으로 확인됩니다.
  • 검침하러 방문하지 않아도 되고, 매달 카운터를 직접 알려주실 필요가 없습니다.
PC가 켜져 있어야 수집됩니다 프린터와 같은 네트워크에 있는 PC 한 대에서 실행해두시면 됩니다.
