사용안내 — 접수관리툴 사용법

전체 흐름 한눈에 보기

① 신규접수 입력▶ ② 접수내역에서 확인▶ ③ 상세에서 처리내용 입력▶ ④ 상태를 완료/출고로 변경

접수 한 건은 위 4단계로 끝납니다. 임대 관련 접수는 ④에서 완료 또는 출고가 되는 순간 임대관리의 수리내역에 자동으로 기록됩니다(아래 4번 항목 참고).

1. 신규접수 넣기

상단 메뉴에서 신규접수를 누릅니다.

  1. 고객 성명 — 회원 검색 버튼으로 기존 고객을 찾아 선택하세요. 이름·휴대폰·전화·주소로 검색됩니다. 새 고객이면 직접 입력해도 됩니다.
  2. 상품 — 상품 검색(전체) 또는 임대제품(임대 항목만) 버튼으로 고릅니다. 고르면 상품 코드·상품명이 자동으로 채워집니다.
  3. 시리얼번호 — 장비 시리얼이나 모델명을 적습니다.
  4. 요청방법 — 출장 / 택배 / 내방 중 선택합니다.
  5. 고객 요청사항 — 고객이 말한 증상이나 요청을 그대로 적어두면 처리할 때 편합니다.
  6. 다 채웠으면 저장합니다.
처리자는 비워두셔도 됩니다 비워두면 지금 로그인한 본인 성명이 자동으로 들어갑니다. 다른 사람이 처리할 건이면 그 사람 이름을 적으세요.

2. 접수내역에서 찾기

상단 메뉴 접수내역에서 전체 접수를 봅니다. 상태 탭을 누르면 그 상태만 걸러서 볼 수 있습니다.

전체접수진행센터 견적택배완료출고

목록에서 건을 클릭하면 상세 화면으로 들어갑니다.

3. 처리하고 상태 바꾸기

  1. 접수내역에서 해당 건을 클릭합니다.
  2. 처리 내용 칸에 작업한 내용을 적습니다.
  3. 상태를 알맞게 선택합니다.
  4. 저장을 누릅니다.
이전 내용은 지워지지 않습니다 저장하면 07월 19일 14시 30분 홍길동 기사의 의견 : 완료 같은 형식으로 기존 내용 아래에 계속 쌓입니다. 그동안의 처리 이력이 그대로 남습니다.
출고로 저장하면 수정이 잠깁니다 상태를 출고로 저장하면 그 건은 더 이상 고칠 수 없습니다. 내용을 다 정리한 다음 마지막에 출고로 바꾸세요.

택배택배로 발송/회수 처리
완료작업이 끝난 상태

4. 임대 건 접수 시 꼭 알아둘 것 (중요)

임대 관련 접수는 완료 또는 출고가 되면 임대관리 → 임대거래처의 수리내역에 자동으로 올라갑니다. 따로 옮겨 적을 필요가 없습니다.

접수한 상품명어디에 기록되나금액
임대출장무상 수리내역출장비 30,000원이 지출로 자동 기록
임대초기설치무상 수리내역0원 (필요하면 임대관리에서 입력)
임대제품교체무상 수리내역0원 (필요하면 임대관리에서 입력)
임대(소모품)판매유상 판매수리내역0원으로 들어가니 임대관리에서 판매금액을 입력하세요

자동으로 올라가려면 아래 3가지가 맞아야 합니다.

  1. 상품명을 위 4가지 중에서 정확히 고를 것 — 반드시 임대제품 버튼으로 선택하세요. 직접 타이핑하면 글자가 조금만 달라도 기록되지 않습니다.
  2. 상태가 완료 또는 출고일 것 — 접수·진행 상태에서는 아직 올라가지 않습니다.
  3. 고객 성명이 임대거래처에 등록된 상호와 같을 것 — 등록되지 않은 거래처면 기록되지 않습니다. (주)·㈜·주식회사, 띄어쓰기 차이는 자동으로 맞춰줍니다.
임대제품회수 등 나머지 임대 항목은 자동기록되지 않습니다 위 표의 4가지만 대상입니다. 그 외 항목은 접수 기록으로만 남습니다.

5. 이럴 땐 이렇게 (자주 묻는 질문)

증상확인할 것
임대 수리내역에 안 올라와요 ① 상태가 완료/출고인지 ② 상품명이 임대 4가지 중 하나로 정확히 들어갔는지 ③ 고객 성명이 임대거래처에 등록된 상호인지 — 이 순서로 확인하세요.
유상 판매인데 금액이 0원이에요 정상입니다. 금액은 임대관리 → 임대거래처의 해당 내역에서 직접 입력하시면 됩니다.
접수를 잘못 저장했어요 상세 화면에서 고쳐 저장하면 됩니다. 단 출고 상태면 잠겨서 수정이 안 되니 관리자에게 문의하세요.
처리자가 아이디로 나와요 사용자 정보에 성명이 비어 있는 경우입니다. 관리자가 엔지니어(사용자) 관리에서 성명을 넣어주면 이후부터 이름으로 표시됩니다.
비밀번호를 바꾸고 싶어요 상단 메뉴 비밀번호변경에서 바로 바꿀 수 있습니다.
